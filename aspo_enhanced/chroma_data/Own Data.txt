How to Add Your Own Data to the Knowledge Base

Step 1: Create a folder named "knowledge" in the project directory if it does not already exist.

Project Structure:
project/
│
├── chatbot.py
├── ingest.py
├── knowledge/
│   ├── file1.txt
│   ├── file2.txt
│   └── your_file.txt
│
└── chroma_db/

Step 2: Add one or more .txt files into the knowledge folder.

Example:
knowledge/company_info.txt
knowledge/training_notes.txt
knowledge/python_basics.txt

Step 3: Put your information inside the text files.

Example content of company_info.txt:

Our company was founded in 2020.
We provide AI and software development services.
Our headquarters are in Bengaluru.

Step 4: Run the ingestion script:

python ingest.py

The script will:
1. Read all .txt files from the knowledge folder.
2. Convert the text into vector embeddings.
3. Store the vectors in ChromaDB.

Step 5: Start the chatbot:

python chatbot.py

Step 6: Ask questions related to your uploaded data.

Example:
Question: Where is the company headquarters?
Answer: Bengaluru.

Important Notes:
- Only .txt files are supported by default.
- After adding new files, run ingest.py again.
- If a file is modified, re-ingest the knowledge base.
- Large files should be split into smaller sections for better retrieval.
